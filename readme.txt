30、应用备份漏洞检查

其他:恶意sdk&广告sdk等。
0x2 使用方法：

命令行参数：
args:
	--taskpath [apkpath]
	--output json/html

examples:
	python AndroidCodeCheck.py --taskpath [path to apk] --output json

0x3 报告输出
报告输出路径在report下
1、json格式
结果以json格式输出，方便和其他的系统集成。
2、html格式
请使用浏览器查看。
